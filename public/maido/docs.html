Maido Features and documentation

Introduction

Maido is a task list tool. It was designed to be "as simple as you need and as powerful as you want"; to make it both intuitive and extensible.

What does this mean for you? Each task has 3 boxes and a description; and features will only manifest when you need them to. For example, you can type a lot of things into the date bar with cool use cases, but you'll probably often just use the date function.

Furthermore, all that documentation down there? Is probably going to be interesting but may not be super practical for you all the time. Remember, Maido is meant to be intuitive. If you're ever feeling stuck, just do what seems natural. Maido won't hate you for breaking stuff.

How to use

To enter a new task, type a task name, date, tag or anything really into the empty row below "Add a new task". Press Enter to add the task.

Loading and saving

To save, press Ctrl-S. Maido will automatically load your last saved task list when you open it.

Use quickload to load the last saved version. To load previous versions, use File->Load.

By default, Maido does not autosave your task list. Autosave can be activated in the Preferences menu.

Dates

In the date box of each task, you can type in a date in a variety of formats. When you press enter, your tasks will be sorted by date.

There are also keywords, which are listed in the table below.

FormatExampleDescription
dd/MM[/yy]12/10/19A date in dd/MM/yy format. If less than 4 digits are provided for yyyy, 2000 years will be added on.
hh:mm[:ss]12:34A time in hh:mm:ss. If a date is not specified and the time specified has passed today, the task will be assigned to tomorrow.
+x(m|h|d|w|M|y)+1wTime to add to the current date in minutes,hours,days, weeks, Months or years. Full names are acceptable. Spaces are not.
doneMarks the task as done, putting it at the bottom of your list.
autoMakes the task time update every time the task list is updated. `auto now` tends to be useful.
nowSets the task time to the current time.
waitingSets the task time 1 day into the future. It was a shortcut I had before I made +1d :3.
somedaySets the task time 1 year into the future. It was a shortcut I had before I made +1y :3.
free:x(m|h|d|w|M|y)free:2hAllocate the next empty space of 2 hours long. It's a little bit trippy right now because unfortunately Maido isn't psychic and doesn't know how long tasks are.
(mon[day]|tue[sday]|wed[nesday]..etc)wedSets the date to the coming weekday; or the next one if this week's weekday has passed. Good for routine things like lectures.

Tags

The tagging system is designed to be extensible. To configure the tags, go to View>Show tag configuration. In that menu enter your tags and formatting (enter to add, as usual.)

The tags literally copy a javascript object into the styling of the associated task, so you might need to know some javascript to edit it. (I'll change that soon.) An example tag could be marker / {"background":"blue"}.

Using tags in your task list is much easier. Simply put #tag_name into the Tags of your selected task, and the formatting will adjust as necessary.

Compain

Compain is Maido's companion. It's designed to be small and unobtrusive and pinned on top of all your other windows, so you can make sure you are on track.

Compain will show you the current and next task on your list. If you have multiple next tasks due before the current time, Compain will turn a mellow shade of yellow to tell you that you should _probably_ get a move on.
